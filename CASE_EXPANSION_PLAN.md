# 掲載企業 100社拡充計画

最終更新: 2026-06-12

## 目的

掲載事例を現在の27社から **100社** に拡充する。

## 選定方針（ユーザー指示）

1. 健康経営優良法人2026の「**ホワイト500**」「**ブライト500**」認定企業から選ぶ
   - 出典: 経済産業省 認定法人一覧（2026-05-12更新版エクセル）
   - ホワイト500: 501社 / ブライト500: 495社
2. 既存掲載27社との**重複は避ける**
   - ホワイト500内の既存掲載: 18社 / ブライト500内: 5社（計23社が認定済み）
3. **健康白書（健康経営白書）を公開している企業を優先**
   - 該当企業は initiatives に itemId "4-2" を必ず入れる

## データ作成ルール（厳守）

- `src/data/cases.js` の既存スキーマに従う（id / companyName / industry / employeeScale / certifications / declaration / promotionStructure / initiatives / kpis / sourceUrl / sourceName）
- **sourceUrl は実在する公式ページ必須**。公開情報のみを使い、創作しない
- 追加した企業は `INDUSTRY_CATEGORY_BY_CASE` に必ず業種大分類を登録する
- categories.js の canSupport 対象12項目は**変更禁止**
- 運輸・空運・鉄道などが増えるため、業種大分類に「運輸・サービス」を追加してよい（INDUSTRY_CATEGORIES への追記）
